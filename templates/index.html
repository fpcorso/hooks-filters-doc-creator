What is this?

This site can analyze a public GitHub repo for a WordPress plugin or theme and will create documenation for all hooks and filters in the plugin or theme.

The documentation will include the hook or filter, accompanying code comments, file and line number for the hook or filter, and a direct link to the line in GitHub.

Fill in the form below with your GitHub username and repo and this site will email you when your documentation is ready.
